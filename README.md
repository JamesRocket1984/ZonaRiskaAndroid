# ЗОНА РИСКА 🚀

Вертикальный космический шутер на Python + Pygame. **25 секторов**, **5 уникальных боссов** на каждом 5-м, **13 типов врагов** с разными атаками, камикадзе из астероидов, мины, бонусы жизни, магазин на станциях, геймпад, синтезированный звук (Dendy/NES + SEGA FM), настраиваемое управление, выбор сложности и таблица из 10 рекордов.

---

## 📂 Файлы проекта
- `main.py` — игра (запускать этот файл)
- `entities.py` — корабли, враги, боссы, отрисовка
- `core.py` — настройки, сохранения, синтез звука
- `requirements.txt` — зависимости
- `run_windows.bat` — запуск на Windows в один клик
- `build_windows.bat` — сборка `ZonaRiska.exe`

## 🎮 Управление (по умолчанию)
| Действие | Клавиатура | Геймпад |
|---|---|---|
| Движение | WASD / стрелки | левый стик / крестовина |
| Огонь | **ЛКМ** / Пробел | A / RB |

| Форсаж | Left Shift | RB |

В паузе: **ESC** — продолжить, **Q** — выйти в меню.

## 🆕 Что нового в v9

- **13 типов противников** — каждый со своей формой, цветом и оружием.
- **5 уникальных боссов** разной формы (клин, кольцо, кристалл, крестовина, многослойный) и с разным оружием. Атаки выбираются **случайно**, а не циклически.
- **Камикадзе из астероидов** — без свечения и без задержки, мгновенный рывок.
- **Меню**: убрана кнопка «Форсаж», добавлен переключатель «Полный экран». Звёзды летят как при форсаже.
- **Пауза** по `ESC` в бою.
- **Без HP-полос** у врагов — больше не мешают.
- **Экономика** перебалансирована: первый босс ×2 опаснее, второй ×5, апгрейды от 3000 cr (×1.5 за уровень).
- **Интенсивность потока врагов зависит от сложности**: лёгкий ×0.375, нормальный ×0.5, сложный ×0.75 (раньше поток был вдвое гуще).
- **Разрушаемые обломки**: спутники, солнечные панели и **обломки станций**. За них не дают кредитов, но из **большого сегмента станции с шансом 50% выпадает жизнь** (у него мигает зелёный маячок). Крупные обломки разлетаются на мелкие.

## 🛠️ Сборка

### Windows: запуск как есть
Двойной клик `run_windows.bat`. Скрипт установит pygame и запустит игру.

### Windows: сборка в `.exe` (отправить другу)
Двойной клик `build_windows.bat`. Создаст `dist\ZonaRiska.exe` (~9 MB). Один файл, ничего не нужно ставить — запускается двойным кликом.

## 🐛 Тонкая балансировка

В `entities.py` параметры врагов задаются массивом `ENEMY_STATS`. Каждая строка:
```python
(тип, мин_сектор, hp_база, скорость, размер, награда_база, hp_на_сектор, награда_на_сектор, атака)
```

HP боссов — массив `BOSS_HP = [800, 3200, 8000, 18000, 40000]`.

Цены магазина — список `SHOP` в `main.py`.
